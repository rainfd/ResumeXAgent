# 技术栈

这是**定义性**的技术选择部分。以下选择是整个项目的唯一真理源 - 所有其他文档必须引用这些选择。

## 云基础设施
- **提供商：** 无（本地部署）
- **核心服务：** 本地文件系统、SQLite数据库、Node.js运行时
- **部署区域：** 本地环境
