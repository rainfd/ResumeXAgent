## 文件夹结构说明

1. **App Router结构：** 使用Next.js 14的App Router，所有路由在app目录下
2. **API路由：** 在app/api下组织RESTful API端点
3. **服务分离：** lib/services包含所有业务逻辑服务
4. **Repository模式：** lib/repositories封装数据访问逻辑
5. **组件组织：** components按功能域分组，ui目录存放shadcn/ui组件
6. **类型安全：** lib/types集中管理TypeScript类型定义
7. **本地数据：** data目录存储SQLite数据库和临时文件
8. **测试组织：** tests目录包含端到端测试和测试数据
